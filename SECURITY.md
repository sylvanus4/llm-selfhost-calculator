# Security & Privacy

This tool is **100% client-side**. There is no backend, no account, and no API key.

- **No data egress.** Every calculation runs in your browser from static JSON files. Nothing you select or type is transmitted anywhere.
- **No tracking, no cookies, no analytics.**
- **Static hosting only.** Served as plain HTML/CSS/JS via GitHub Pages; there is no server to attack.
- **Numbers are estimates.** Model/hardware/price figures are public approximations bundled in `data/*.json`; they may be inaccurate or outdated. Do not use them as the sole basis for spending decisions.

## Reporting

Found an error in a formula or a data value? Open an issue or PR at
https://github.com/sylvanus4/llm-selfhost-calculator/issues — corrections to `data/*.json` and `assets/compute.js` are welcome.
